## Installation Guide

### Step 1: Download and Extract
1. Download the G.A.L package
2. Extract all files to your desired location
3. Keep all files together in the same folder

### Step 2: Install Python (if needed)
- Download Python 3.7+ from [python.org](https://python.org)
- During installation, **check "Add Python to PATH"**
- Verify installation by opening Command Prompt and typing `python --version`

### Step 3: Run the Installer
1. Double-click `Run.bat` to start installation
2. Allow Administrator privileges when prompted
3. The installer will automatically:
   - Check for required dependencies
   - Upgrade pip if needed
   - Install missing packages
   - Launch G.A.L upon completion

### Step 4: Verification
- Check `GAL_Install_Debug.log` for any installation issues
- Ensure G.A.L launches successfully
- Verify all features are accessible

## How to Use G.A.L

### System Optimizer Tab

#### Basic Operations:
1. **Individual Optimizations**:
   - Click any button to run specific optimizations
   - Monitor progress in the status text box
   - Each operation provides detailed feedback

2. **Selective Optimization**:
   - Use checkboxes to select preferred operations
   - Click "Run Preferred" to execute selected optimizations only

3. **Complete Optimization**:
   - Click "Run All" for comprehensive system optimization
   - Operations run in parallel for efficiency

#### Available Optimizations:
- **RAM Cache Cleaning** - Free up system memory
- **Temp File Cleaning** - Remove temporary files from multiple locations
- **Prefetch Cleaning** - Clear application prefetch data
- **Recycle Bin** - Permanently empty recycle bin
- **DNS Management** - Flush cache and set optimal DNS
- **Power Plans** - Configure high-performance power schemes
- **Windows Services** - Disable telemetry, Cortana, Game Bar
- **Network Optimization** - Optimize adapter settings for gaming

### Game Optimizer Tab

#### Setting Up Game Monitoring:
1. **Add Programs to Monitor**:
   - **Manual Entry**: Type program name (e.g., `game.exe`) and click "Add Program"
   - **Directory Scan**: Click "Add Directory" to scan folders for .exe files
   - **Load List**: Import saved program lists from file

2. **Save Your Configuration**:
   - Use "Save List" to store your program list for future sessions
   - Load previous lists with "Load List"

#### Power Plan Management:
1. **View Available Plans**:
   - Click "Refresh Plans" to see all power schemes
   - Active plan is marked with [ACTIVE]

2. **Manage Power Plans**:
   - **Set Active**: Select a plan and click "Set Active"
   - **Delete Plan**: Remove unwanted plans (cannot delete active)
   - **Create New**: Generate custom plans from templates

3. **Trickster Feature** (Tools Menu):
   - Unlocks hidden "Ultimate Performance" plan
   - Disguises it as "High Performance" for easy selection
   - Automatically activates the optimized plan

#### Process Monitoring:
1. **Start Monitoring**:
   - Click "Start Monitoring" to begin watching for programs
   - G.A.L will automatically optimize detected games
   - Monitor status in the log window

2. **Manual Process Optimization**:
   - **Select Process**: Choose from running processes dropdown
   - **CPU Affinity**: Set specific cores (e.g., "0,1,2,3")
   - **Priority Level**: Choose from available priority classes
   - **Apply**: Click "Apply Process Optimization" to implement settings
   - **Remove**: Reset to defaults with "Remove Process Optimization"

3. **Stop Monitoring**:
   - Click "Stop Monitoring" when finished gaming
   - System returns to normal operation

### Tools Menu

#### Available Tools:
- **Clear Log** - Clear current tab's log content
- **Save Log** - Export log to text file
- **Set Window Size** - Adjust application window dimensions
- **Debug Console** - View detailed debug information
- **Trickster** - Activate power plan manipulation feature
- **Exit** - Close the application

#### Window Size Settings:
- Pre-defined common resolutions
- Custom resolution input
- Instant application resizing

#### Debug Console:
- Real-time debug log viewing
- Refresh to see latest entries
- Essential for troubleshooting

## Troubleshooting

### Common Issues:

1. **Administrator Privileges Required**:
   - Right-click G.A.L and "Run as Administrator"
   - Ensure User Account Control allows changes

2. **Python Not Found**:
   - Reinstall Python with "Add to PATH" enabled
   - Verify installation with `python --version`

3. **Missing Dependencies**:
   - Run `Run.bat` to auto-install requirements
   - Check `GAL_Install_Debug.log` for specific errors

4. **Features Not Working**:
   - Verify Administrator privileges
   - Check Windows version compatibility
   - Review debug console for error messages

### Support Resources:
- Check `debug_log.txt` for detailed operation logs
- Use Debug Console in Tools menu for real-time monitoring
- Review installation log for setup issues
