G.A.L - Gamers As Legions
=========================

A Comprehensive System Optimization Tool for Gamers

OVERVIEW
--------
G.A.L (Gamers As Legions) is an all-in-one system optimization tool designed specifically for gamers and power users. This application provides comprehensive system cleaning, real-time performance monitoring, game optimization, and power management features to maximize your PC's performance.

KEY FEATURES
------------

1. SYSTEM OPTIMIZER
   • Clean RAM cache and optimize memory usage
   • Remove temporary files from multiple locations
   • Clean browser caches and internet files
   • Flush DNS cache and optimize network settings
   • Clean Windows update files and delivery optimization
   • Disable telemetry and unnecessary services

2. GAME OPTIMIZER
   • Monitor and automatically optimize running games
   • Set CPU affinity and process priority for games
   • Power plan management for maximum performance
   • Create custom power plans from templates
   • Process monitoring and automatic optimization

3. PERFORMANCE MONITOR
   • Real-time system performance tracking
   • CPU, memory, disk, and network monitoring
   • GPU temperature and usage monitoring
   • Enhanced system information with detailed hardware specs
   • Network speed monitoring in real-time

4. ADVANCED SYSTEM INFORMATION
   • Comprehensive hardware and software analysis
   • Python and CUDA environment details
   • Storage drive type detection (SSD/HDD)
   • Network adapter information and USB devices
   • BIOS, motherboard, and DirectX details

5. SAFE MODE
   • Toggle safe mode to restrict system modifications
   • Skip DNS changes, registry modifications, and power plan changes
   • Only perform file cleaning operations in safe mode
   • Perfect for troubleshooting or cautious users

INSTALLATION
------------

1. Ensure you have Python 3.7 or higher installed
2. Run the application as Administrator for full functionality
3. The application will automatically install required dependencies

SYSTEM REQUIREMENTS
-------------------
• Windows 10 or Windows 11 (64-bit recommended)
• Python 3.7 or higher
• Administrator privileges for full functionality
• Minimum 4GB RAM (8GB recommended)
• 100MB free disk space

USAGE INSTRUCTIONS
------------------

1. SYSTEM OPTIMIZATION:
   - Go to System Optimizer tab
   - Select desired optimizations using checkboxes
   - Click "Run Preferred" for selected options or "Run All" for complete optimization
   - Use Safe Mode for limited operations

2. GAME OPTIMIZATION:
   - Add games to monitor in Game Optimizer tab
   - Use Power Plan Management to set high-performance plans
   - Start monitoring to automatically optimize detected games
   - Use manual process optimization for specific applications

3. PERFORMANCE MONITORING:
   - Start monitoring in Performance Monitor tab
   - View real-time system statistics
   - Use Advanced System Info for detailed hardware analysis

4. POWER PLAN MANAGEMENT:
   - Available in Game Optimizer tab
   - Set active power plans for maximum performance
   - Create custom power plans from templates
   - Restore default power plans if needed

SAFETY FEATURES

• DNS changes include backup and restore functionality
• Confirmation dialogs for critical operations
• Comprehensive logging for troubleshooting
