G.A.L - Gamers As Legions
=========================

A Comprehensive System Optimization Tool for Gamers

OVERVIEW
--------
G.A.L (Gamers As Legions) is an all-in-one system optimization tool designed for gamers and power users who want cleanup, optimization, monitoring, game-focused tuning, and advanced Windows utilities in one application. The current build includes tabs for System Optimizer, Game Optimizer, Performance Monitor, and Help, along with a Tools menu for advanced actions such as Debug Console access, DNS restore, polling-rate actions, priority tools, standard-settings restore, and reservable-bandwidth controls.

KEY FEATURES
------------

1. SYSTEM OPTIMIZER
   • Clean RAM cache and review high-memory processes
   • Remove temporary files from multiple system locations
   • Clean browser caches and temporary internet files
   • Flush DNS cache and manage DNS optimization actions
   • Clean Windows Update cache and Delivery Optimization files
   • Clean Defender scan files, Windows upgrade logs, and system recovery logs
   • Disable Telemetry, Cortana, and Game Bar
   • Optimize network adapter settings
   • Use Run Preferred for selected actions or Run All for batch optimization

2. GAME OPTIMIZER
   • Monitor and optimize selected games and applications
   • Set CPU affinity and process priority for running processes
   • Power plan management for performance-focused usage
   • Create custom power plans from built-in templates
   • Refresh, activate, delete, and restore power plans
   • Process monitoring and manual optimization tools

3. PERFORMANCE MONITOR
   • Real-time system performance tracking
   • CPU, memory, disk, and network monitoring
   • In-app performance text output and status review
   • Enhanced utility access for ongoing troubleshooting and testing

4. ADVANCED SYSTEM INFORMATION
   • Summary-style hardware and software analysis
   • Storage, network, OS, memory, and Python environment reporting
   • Exportable system information from the Advanced System Info tool
   • Additional diagnostics available through the Tools menu

5. SAFE MODE
   • Toggle Safe Mode to restrict system modifications
   • Skip sensitive changes when cautious operation is preferred
   • Helpful for testing, troubleshooting, or limited cleanup sessions

6. TOOLS MENU
   • Clear Log
   • Save Log
   • Advanced System Info
   • Set Window Size
   • Debug Console
   • Disable Limit Reservable Bandwidth
   • Revert LRB
   • Restore Original DNS
   • Optimize Polling Rate
   • Revert Polling Rate
   • Priority Bypasser
   • Restore Standard Windows Settings
   • Trickster
   • Exit

7. DEBUG AND LOGGING
   • Debug Console includes Refresh and Clear controls
   • Clear wipes debug.log and debug_log.txt for a fresh session
   • Cleanup actions now summarize deleted and skipped locked/protected items
   • Logging helps with testing, troubleshooting, and repeat runs

INSTALLATION
------------

1. Ensure Python 3.7 or higher is installed
2. Run the application as Administrator for full functionality
3. The application can install required dependencies automatically
4. Use the latest script build to access the newest Tools menu options and fixes

SYSTEM REQUIREMENTS
-------------------
• Windows 10 or Windows 11 (64-bit recommended)
• Python 3.7 or higher
• Administrator privileges for full functionality
• Minimum 4GB RAM (8GB recommended)
• Adequate free disk space for cleanup and logs

USAGE INSTRUCTIONS
------------------

1. SYSTEM OPTIMIZATION:
   - Go to the System Optimizer tab
   - Select desired optimizations using checkboxes
   - Click "Run Preferred" for selected options or "Run All" for full optimization
   - Review the status box for deleted/skipped summaries and results
   - Use Safe Mode for more limited operation

2. GAME OPTIMIZATION:
   - Add games or applications to monitor in the Game Optimizer tab
   - Use Power Plan Management to set or create performance-focused plans
   - Start monitoring to automate optimization behavior
   - Use manual process optimization for specific applications

3. PERFORMANCE MONITORING:
   - Start monitoring in the Performance Monitor tab
   - View real-time system statistics
   - Use Advanced System Info from the Tools menu for more details

4. TOOLS MENU ACTIONS:
   - Open Debug Console for logs and quick clearing
   - Restore Original DNS when needed
   - Use polling-rate actions and Priority Bypasser from the Tools menu
   - Use Disable Limit Reservable Bandwidth to create/verify the Psched policy key and set NonBestEffortLimit to 0
   - Use Revert LRB to remove NonBestEffortLimit and restore default Windows behavior

SAFETY FEATURES
---------------
• Safe Mode restricts system modifications
• DNS changes include backup and restore behavior
• Warning prompts can alert users that open programs may need to close before cleanup succeeds fully
• Comprehensive logging helps with troubleshooting
• Cleanup skips locked or protected files more gracefully instead of flooding the UI with repetitive errors
